# Core

App core in rust

## Dependencies

### ORM - Sea ORM

https://www.sea-ql.org/SeaORM/

CLI is mandatory to interract with migration

```bash
cargo install sea-orm-cli
```

https://docs.rs/shaku/latest/shaku/guide/index.html
